# React UI组件库
### 安装
~~~javascript
npm install freemyui --save
~~~
### 使用
~~~javascript
// 加载样式
import 'freemyui/dist/index.css'
// 引入组件
import { Button } from 'freemyui'
~~~
###  相关技术
* 🔥typescript with React Hooks
* ⛑️使用 react-testing-library 完成单元测试
* 📚使用 storybook 本地调试和生成文档页面

### 一些本地开发命令

~~~bash
//启动本地环境
npm run stroybook

//跑单元测试
npm test

//build可发布静态文件
npm run build

//发布到 npm
npm run publish
~~~
